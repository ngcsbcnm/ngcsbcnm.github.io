Linux常用文件管理命令

2023/2/20 Linux

取消命令并换行

 ctrl c

清空本行命令

 ctrl u

补全命令和文件名

tab键

显示目录下的文件

ls #列出当前目录下所有文件，蓝色的是文件夹，白色的是普通文件，绿色的是可执行文件

显示当前路径

pwd

进入目录

cd XXXX #进入XXXX目录下，cd .. 返回上级目录

复制文件

cp XXX YYY #将XXX文件复制成YYY，XXX和YYY可以是路径，比如../dir_c/a.txt，表示上层目录下的dir_c文件夹下的文件a.txt

创建目录

mkdir XXX #创建目录XXX

删除文件

rm XXX #删除普通文件;  rm XXX -r: 删除文件夹

移动文件

mv XXX YYY #将XXX文件移动到YYY，和cp命令一样，XXX和YYY可以是一个路径；重命名也是用这个命令

创建文件

 touch XXX  #创建一个文件

展示文件

 cat XXX #展示文件XXX中的内容

复制文本

windows/Linux下：Ctrl + insert，Mac下：command + c

粘贴文本

  windows/Linux下：Shift + insert，Mac下：command + v
